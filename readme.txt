Инициализация криптопровайдера.
1. bool Init(String container, String provider)

Возвращаемое значение логического типа (истина - успех, ложь - неудача)
Параметры
	- container (строковый тип, наименование контейнера для хранения ключей)
	- provider (строковый тип, наименование криптопровайдера, есть 3 варианта
				наверное стоит использовать
				"Crypto-Pro GOST R 34.10-94 Cryptographic Service Provider")

Создание ключевой пары.
2. bool CreateNewKeys()

Возвращаемое значение логического типа (истина - успех, ложь - неудача)
Параметры - нет

Экспорт открытого ключа
3. String ExportPublicKey()

Возвращаемое значение строка открытый ключ в шестнадцатиричном виде
Параметры - нет

Подпись строки
4. String SignMessage(String msg)

Возвращаемое значение строка подпись в шестнадцатиричном виде
Параметры
	- msg (строковой тип, подписываемое сообщение)

Проверка подписи
5. bool VerifyMessage(String msg, String sign, String key)

Возвращаемое значение логического типа (истина - успех, ложь - неудача)
Параметры
	- msg (строковой тип, проверяемое сообщение)
	- sign (строковой тип, подпись)
	- key (открытый ключ)
