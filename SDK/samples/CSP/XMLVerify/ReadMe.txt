Пример использует результат работы программы XMLSignForVerify.exe.
Соответственно подразумевается, что соблюдены рекомендации по работе
с примером XMLSignForVerify.

Запуск: 
в командной строке нужно задать имя подисанного xml-документа,
полученнного после отработки XMLSignForVerify.exe.
Предварительно этот файл нужно скопировать в дирректорию проекта,
в папку Debug (там же, где и exe-файл).
