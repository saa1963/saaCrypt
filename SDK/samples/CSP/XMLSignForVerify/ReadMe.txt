Инструкция

Для того, чтобы пример работал, необходимо выполнить следующие действия:

1. Установить Word 2003: XML Viewer
2. Выпустить сертификат на тестовом центре сертификации,
   расположенном по адресу:

http://www.cryptopro.ru/certsrv/ 

Примечания.

- При заполении формы для формирования запроса на сертификат в поле Name
    указывается то значение, которое затем передаётся в качестве второго
    параметра в командной строке. Под таким именем сертификат
    устанавливается в хранилище "Личные" текущего пользователя.

- Рекомендуется пометить галочку "Задать имя ключевого контейнера"
  ("Set the container name") и задать осмысленное имя для ключевого
  контейнера для облегчения выполнения дальнейших операций.
  
- Установить сертификат на компьютер, при этом сертификату будет сопоставлена
  ссылка на секретный ключ. Если этого не произошло, но у вас есть сертфикат
  в виде файла и секретный ключ, то можно выполнить пункт 3 (если установка
  сертификата прошла без ошибок, то выполнять его не надо)

3. Связать существующий сертификат из файла с существующим
   контейнером секретного ключа на носителе. Для этого:

  1) Скопировать выпущенный сертификат в файл (при просмотре сертификата
   вкладка "Состав", кнопка "Копировать в файл...");

  2) открыть панель КриптоПро CSP:
     Пуск/Настройки/Панель управления/КриптоПро CSP;

  3) Во вкладке "Сервис" воспользоваться кнопкой "Установить личный сертификат"
   и далее следовать указаниям мастера.

//----------------------------------------------------
Запуск в командной строке:

XMLSignForVerify.exe doc_to_sign.xml certName

где  doc_to_sign.xml - xml-документ, который нужно подписать
        certName - имя сертификата (а не имя файла сертификата!)
        в хранилище "Личные" текущего пользователя
